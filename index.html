HTML itu adalah sebuah bahasa yang menggunakan markup atau penanda untuk membuat halaman web. Penanda atau markup ini, nanti akan kita sebut dengan Tag. HTML berperan untuk menentukan struktur konten dan tampilan dari sebuah web. HTML itu seperti batu bata untuk membangun rumah. Batu bata ini dapat disusun, hingga menjadi fondasi dasar.

  1. Tag adalah sebuah penanda awalan dan akhiran dari sebuah elemen di HTML. Tag dibuat dengan kurung siku (<...>), lalu di dalamnya berisi nama tag dan kadang juga ditambahkan dengan atribut.
  2. Elemen dalam HTML adalah sebuah komponen yang menyusun dokumen HTML. Elemen kadang juga disebut sebagai node, karena ia merupakan salah satu jenis node yang menyusun dokumen HTML dalam diagram HTML tree.
  3. Atribut adalah kata khusus yang berada di dalam tag pembuka. Atribut juga disebut sebagai modifier yang akan menentukan perliaku dari elemen.

Pada materi ini, kita belajar beberapa tag:

  1. <p> untuk membuat paragraf
  2. <hr> untuk membuat garis lurus
  3. <br> untuk membuat baris baru
  4. <div> untuk membuat paragraf di luar artikel atau layout
  5. <pre> untuk membuat paragraf dengan format yang sudah ditentukan.

Ini adalah tag-tag dasar yang perlu diingat dalam membuat paragraf di HTML.

Heading adalah sebuah judul yang biasanya diberikan pada halaman atau beberapa bagian dari artikel. Jika kamu sering menulis artikel, pasti ini tidak asing buatmu.

Komentar adalah elemen yang akan diabaikan oleh browser. Ia tidak akan ditampilkan di dalam web. Komentar biasanya digunakan untuk memberikan informasi tambahan pada kode HTML dan kadang untuk membantu developer lain memahami alur kerja dari HTML yang kita buat.

Tag HTML untuk membuat heading ada 6:

  • <h1> hingga <h6>
